### Instalação
```
git clone https://github.com/bruno7kp/mural-virtual.git
cd mural
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
set FLASK_ENV='development'
set FLASK_DEBUG=1
python -m flask run
```
